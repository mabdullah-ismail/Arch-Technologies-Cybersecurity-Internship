TCP handshakes:
a 3 step handshake is made everytime our browser connects to a website i.e
Your PC  →  SYN      →  Server  
Your PC  ←  SYN-ACK  ←  Server   
Your PC  →  ACK      →  Server   

Spot a port scan attack:
TCP | 192.168.1.10:1234 -> 192.168.1.1:22
TCP | 192.168.1.10:1234 -> 192.168.1.1:23
TCP | 192.168.1.10:1234 -> 192.168.1.1:80
TCP | 192.168.1.10:1234 -> 192.168.1.1:443
One IP hitting many ports rapidly = someone scanning your network. Security teams use sniffers to detect exactly this.

Understanding DNS:
When you type google.com, your computer first asks "what is Google's IP address?" — this is called a DNS request and uses UDP port 53.
Run your sniffer and open any website. You'll see:
UDP | 192.168.1.5:54321 -> 8.8.8.8:53
That 8.8.8.8 is Google's DNS server. Your computer is asking it to translate google.com into an IP address.
