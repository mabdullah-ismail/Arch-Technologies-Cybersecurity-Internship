Network Sniffer - Task 1
ARCH Technologies - Cyber Security Training

Project Overview
This project implements a basic network sniffer using Python and Scapy. It captures live network traffic, analyzes packet headers, and logs the information for security analysis. This tool demonstrates how network monitoring works and highlights the importance of encryption in network communications.

Objectives
Understand how data flows across networks
Learn packet structure (IP, TCP, UDP, ICMP headers)
Recognize different network protocols
Identify security risks in unencrypted network traffic

Requirements
Python 3.6 or higher
Scapy library (`pip install scapy`)
Administrative/root privileges (required for packet capture)
