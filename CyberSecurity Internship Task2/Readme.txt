### Project Overview
This project implements a basic keylogger to demonstrate how keystroke logging works. It captures keyboard inputs, logs them to a file, and analyzes the captured data. The purpose is to understand the security risks associated with keyloggers and learn how to protect against them.

Objectives
Understand how keyloggers capture keystrokes
Learn about event handling in operating systems
Recognize the security risks of keylogging
Develop awareness of input privacy

Requirements
Python 3.6 or higher
pynput library (`pip install pynput`)
